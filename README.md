# Vitality Med Spa — Mesa, AZ · Redesign Mockup

A single-file HTML mockup proposing a rebuild of **vitalitymedspa.com** that resolves the spa's most urgent problem (expired SSL → browser-blocked homepage) and leads with the differentiator their current site buries: **20 years of low-cost, MD-supervised weight loss + hormone replacement** across three Arizona Valley locations.

- Live: https://vitality-med-spa-mesa.vercel.app
- Demo video: https://vitality-med-spa-mesa.vercel.app/demo.mp4
- Source: https://github.com/yagyesh-bobde/vitality-med-spa-mesa

## What changed vs. the current site

1. **Trust signals replace the SSL warning.** Visible "Secure Site" and "100% MD Supervised" badges sit in the nav and hero — exactly where the current expired-cert warning kills conversion.
2. **Differentiator promoted to section 2.** Weight loss + HRT is not a sub-page link — it is the second screen, framed as "what we're known for."
3. **Real photos throughout.** Hero uses the actual Vitality storefront photo from their own CDN. No stock.
4. **Founder + medical director are named, not anonymous.** Desiree Wooding, Dr. Kristie Blessitt MD, Joanie Vilcheck RN.
5. **On-page booking form.** Name + phone + service + time → success state. Works on file:// without a server.
6. **One unified CTA.** "Book a consult" everywhere.

## Stack

Pure HTML/CSS/vanilla JS — no build step. Fonts: Instrument Serif + DM Sans. Palette: sand / clay / pine (desert wellness).
